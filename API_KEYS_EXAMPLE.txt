# API Keys Configuration Guide

## Getting Your API Keys

### Anthropic Claude API Key

1. Go to https://console.anthropic.com/
2. Sign up or log in to your account
3. Navigate to "API Keys" section
4. Click "Create Key"
5. Copy your key (starts with "sk-ant-...")
6. Paste it into TocoAce Settings > Anthropic API Key field

**Pricing**: Pay as you go
- Claude 3.5 Sonnet: ~$3 per million input tokens, ~$15 per million output tokens
- Typical interview answer: ~$0.01-0.05 per generation

### OpenAI API Key

1. Go to https://platform.openai.com/
2. Sign up or log in
3. Navigate to "API Keys" in your account settings
4. Click "Create new secret key"
5. Copy your key (starts with "sk-...")
6. Paste it into TocoAce Settings > OpenAI API Key field

**Pricing**: Pay as you go
- GPT-4o: ~$5 per million input tokens, ~$15 per million output tokens
- GPT-4 Turbo: ~$10 per million input tokens, ~$30 per million output tokens
- Typical interview answer: ~$0.02-0.10 per generation

## Troubleshooting

### "API key not configured" error
- Make sure you've entered the key in Settings
- Check that the key starts with the correct prefix
- Verify there are no extra spaces

### "Authentication failed" error
- Key may be invalid or expired
- Regenerate your key in the respective console
- Check your account has active billing

### "Rate limit exceeded" error
- You've made too many requests too quickly
- Wait a few minutes and try again
- Consider upgrading your API plan

### "Insufficient credits" error
- Add more credits/payment method to your account
- Check your billing status in the console

## Multiple API Keys (Advanced)

You can have both Anthropic and OpenAI keys configured:
- Switch between engines in Settings
- This gives you flexibility and redundancy
- Useful if one service is down or rate-limited
